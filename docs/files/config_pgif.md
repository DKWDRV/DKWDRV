## All PGIF internal configs used in DKWDRV. 

Compiled from all regions of the consoles. 
Note that each PGIF console only has the configs for the respective region it was released, meaning any PGIF console running on default PS1DRV would fail to apply game configs for discs of other regions (using modchips, swap method, PS1VmodeNeg etc..)
DKWDRV has support for all games available.
All values are in decimal (including mecha). 
<br>
Mecha values might do slightly differently in PGIF mecha than in newer Dragon mecha (50k+ models.)
Since PGIF models have both old and Dragon mecha, for some games mecha config was changed. 

Sometimes other parts of the config were changed too! You can find a full list of these games in [here](https://github.com/DKWDRV/DKWDRV/blob/main/docs/files/config_pgif_2.md).

Speedrunning community should not be surprised at these different results for different consoles! Sometimes it was not applied config at all, sometimes the different config, sometimes the change in mechacon, sometimes the mecha dsp and laser config in eeprom and a lot of other factors.
Japanese consoles apply some type of different emulation in mecha firmware for CD audio (unclear what and why)
